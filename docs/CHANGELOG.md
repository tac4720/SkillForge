# Changelog

## 0.1.0

- Bootstrapped the TypeScript, Vitest, and Playwright workspace
- Added pure core implementations for permission policy, path sanitizer, input validation, redaction, risk classification, normalizer, parameterizer, and assertion evaluation
- Added contract definitions plus deterministic fake browser, shell, filesystem, approval, and secret adapters
- Implemented replay engine, run logger, repair engine, and local registry
- Implemented local daemon, browser recorder, native package exporter, OpenClaw exporter, and CLI command dispatcher
- Added local fixture-backed integration, security, and Playwright E2E coverage
- Added baseline project documentation and lint configuration
